# Cloud Infrastructure Automation Using Terraform on AWS

## Overview
This project automates AWS infrastructure provisioning using Terraform with remote state management and state locking.

## Features
- Infrastructure as Code (IaC)
- AWS EC2 provisioning
- Terraform modules
- S3 backend
- DynamoDB state locking
- Reusable infrastructure

## Technologies Used
- Terraform
- AWS
- S3
- DynamoDB
- EC2
- GitHub

## Architecture
Terraform → AWS Infrastructure → Remote State Management
